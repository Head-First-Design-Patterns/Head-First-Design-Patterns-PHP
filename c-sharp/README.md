# Head First Design Patterns in PHP
##Someone has already solved your problems

These patterns are ported from the Java examples in the O'Reilly book ["Head First Design Patterns"](http://www.headfirstlabs.com/books/hfdp/).

* [Strategy Pattern](StrategyPattern/StrategyPattern/Strategy.cs) - Defines a family of algorithms, encapsulates each one, and makes them interchangeable. Strategy lets the algorithm vary independently from clients that use it.

##OO Basics
* Abstraction
* Encapsulation
* Polymorphism
* Inheritance

##OO Principles
* Encapsulate what varies
* Favor composition over inheritance
* Program to interfaces, not implementations
